# 📊 Mental Health in Tech Analysis (DS.v3.2.1.5)

Data and statistical analysis, and visualization of Mental Health in Tech in late 2010s. Data is taken from years 2014 up to 2019 from  [Open Source Mental Illness (OSMI) on Kaggle](https://github.com/TuringCollegeSubmissions/cgarci-DS.v3.2.1.5)

Observing data on different tech roles, company size, diagnosed conditions and how they relate to each other.


## 📔 Notebook Contents
- Demographical analysis 
- Diagnosis Counts and Prevalence Rates
    - Among Different Tech Roles
- Correlations and Heatmap
- Suggestions for Further Research
- Conclusions

## 📦 Packages and Libraries Used:
- numpy 
- pandas
- matplotlib
- seaborn
- plotly
